# LPC Sprite Attribution

The character sprite layers in this directory are derived from the
**Universal LPC Spritesheet** (Liberated Pixel Cup).

- Source: https://github.com/jrconway3/Universal-LPC-spritesheet
- License: **CC-BY-SA 3.0** and **GPL 3.0** (dual-licensed).

These assets require attribution and share-alike. See the upstream repository
for the full list of original authors and per-asset credits.

Each `*.png` here is a single LPC layer sheet (832×1344, 64×64 frames, 13×21
grid). Layers are composited at runtime in z-order: body → pants → shoes →
shirt → hair.
